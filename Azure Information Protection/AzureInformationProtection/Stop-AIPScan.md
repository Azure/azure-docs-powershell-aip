# Stop-AIPScan

## SYNOPSIS

Instructs the Azure Information Protection scanner to immediately stop the currently running scan cycle.

## SYNTAX

```
Stop-AIPScan
```

## DESCRIPTION

## EXAMPLES

### Example 1: Stop the currently running scan cycle

```powershell
PS C:\> Stop-AIPScan
```

## OUTPUTS
